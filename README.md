# FastPlanner

将FastPlanner后端基于B样条的轨迹优化封装单独可用的C++文件（纯净版本），并将三维轨迹将为2维轨迹，适用于地面小车实时规划。

# how to use ?

file 文件下有nlopt安装包，源码编译安装即可.

cd build

cmake ..

make

./main
